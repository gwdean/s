Variance
========

```r
y <- c(13, 7, 5, 12, 9, 15, 6, 11, 9, 7, 12)
var(y)
```

```
## [1] 10.25
```

```r

variance <- function(x) sum((x - mean(x))^2)/(length(x) - 1)
variance(y)
```

```
## [1] 10.25
```

```r

v <- function(x) sum(x/length(x))
v(y)
```

```
## [1] 9.636
```

```r

s <- function(x) sum(x)
s(y)
```

```
## [1] 106
```

```r

s(y)/length(y)
```

```
## [1] 9.636
```
