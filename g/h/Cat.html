1.2 Definition: Point

A point of a set X is a map 1->X.

1.3 Definition: Category

A CATEGORY consists of the DATA:

  • (1) Objects
  • (2) Maps
  • (3) Domain and Codomain
  • (4) Identity Map
  • (5) Composite Map

Satisfying the following RULES:

  • (i) Identity Laws
  • (ii) Associative Law
