Regression

These notes explain how to perform basic regression analysis in R.

Polynomial Regression

# Polynomial Regression
fit2 <- lm(weight ~ height + I(height^2), data = women)
summary(fit2)

## 
## Call:
## lm(formula = weight ~ height + I(height^2), data = women)
##
